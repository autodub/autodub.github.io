Speech Denoising using Wavenet


Currently, most speech processing techniques use magnitude spectrograms as frontend and apply some transformation techniques to filter some features(generally termed as noise) from that spectrogram. Combination of Mel-Spectrograms & Wavenet's performance on speech signals inspired us. So we decided to use this duo to filter out noise from speech.
1. Replaced magnitude spectrogram with Mel-Spectrogram.
2. Replaced Fourier filters with Wavenet.
